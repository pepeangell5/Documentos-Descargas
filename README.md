Repositorio dedicado a compartir **guías, manuales, diagramas y documentos técnicos** relacionados con mis dispositivos, proyectos y prototipos electrónicos.

La idea de este espacio es mantener la documentación organizada para que cualquier persona pueda consultarla, descargarla y usarla como referencia en sus propios proyectos.

---

## 📌 Contenido del repositorio

Aquí se irán agregando documentos como:

* 📘 Guías de construcción
* 🔌 Diagramas de conexión
* 🧩 Manuales de uso
* 🛠️ Documentación técnica
* 📄 Archivos PDF descargables
* ⚙️ Notas de configuración
* 🧪 Pruebas y referencias de laboratorio

---

## 📂 Organización

Los documentos estarán separados por carpetas según el dispositivo, módulo o proyecto correspondiente.

```bash
/
├── Proyecto-1/
│   ├── Guia-de-construccion.pdf
│   ├── Diagrama-de-conexiones.png
│   └── Manual-de-uso.pdf
│
├── Proyecto-2/
│   ├── Documentacion-tecnica.pdf
│   └── Notas-de-configuracion.md
│
└── README.md
```

---

## 🎯 Objetivo

Crear una biblioteca pública de documentación para facilitar el aprendizaje, la construcción y la consulta de información técnica sobre dispositivos basados en microcontroladores, módulos electrónicos y proyectos DIY.

---

## ⚠️ Aviso

Todo el contenido compartido en este repositorio tiene fines **educativos, informativos y de documentación**.

Cada usuario es responsable del uso que le dé a la información, conexiones, códigos o dispositivos descritos en los documentos.
